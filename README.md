# CulturePulse

CulturePulse is a small, static site celebrating Rwandan culture through storytelling, community programs, and artistic highlights.

## Structure

* `index.html` – Landing page with hero, about, projects, and skills.
* `mission.html` – Mission and focus areas.
* `voice.html` – Stories, poetry, dance, and music highlights.
* `contact.html` – Contact form.
* `style.css` – Global styles and layout.
* `assets/` – Images used across pages.
* `demo.mp4` – Optional preview media.
